# Laborativo_Quicklink

Landing pública de Laborativo — link rápido que se entrega a posibles clientes con info de la compañía, esencia y casos de éxito.

## Stack
- Next.js 16 + TypeScript + Tailwind v4
- Deploy en Vercel
- Sin Supabase ni backend por ahora (contenido estático)

## Convenciones
- App Router, `src/` dir, alias `@/*`
- Server components por defecto, client components solo cuando hay interactividad
- Diseño aún por definir — no asumir paleta ni tipografía hasta que se acuerde
